#  Hand Detection With Python

### - Hand Detection  ---> hand_detect.py

###  - Hand Finger Counter --> finger_counter.py

###  - Hand Vluome Control --> hand_volume_ctr.py


- **description:**  A simple **Python** project in the field of computer vision. Hand recognition, finger counting, volume control with hand movement and module writing...

#### Dependencies :
install **opencv** and **mediapipe** library with ``` pip install opencv-python``` and ``` pip install mediapipe ``` command.

- **You can also install dependencies via the following command:**
- ‍‍``` pip install -r requirements.txt```
